Texts in template files

Files per Table template file

  • Use {table} in filename to replace table's name
  • Between {foreach field} and {/fieldforeach} will be repeated for each field
    • {[field.name]}
    • {[field.label]}
    • {[field.description]}
    • {[field.default_value]}
    • {[field.lang.*]} = replace * with language name, {[field.lang.Java]} would replace Java named value, remember correct lettercase.
    • {[field.auto_increment]}, {[field.not_null]}, {[field.unique]} and {[field.primary_key]} fields, if true, will be replaced with values that are set in Miscellaneous settings, otherwise would replaced with nothing
  • Between {foreach foreignkey} and {/foreignkeyforeach} will be repeated for each foreign key, most likely you would use this in SQL queries
    • {[key.name]}
    • {[key.field.name]}
    • {[key.field.lang.*]} = replace * with language name, {[field.lang.Java]} would replace Java named value, remember correct lettercase.
    • {[key.foreign.table]} foreign table's name
    • {[key.foreign.field]} foreign table's field's name. {[key.foreign.table]}({[key.foreign.field]}) would be replaced like Users(user_id)

Many Tables, One File template file

  • Same as Files per Table template file, but use {foreach table} and {/tableforeach} to go thru selected tables.
